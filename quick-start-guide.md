# Quick Start Guide

## What this tracker does

This Google Sheets and Excel tracker helps personal trainers manage:

- Clients
- Session packages
- Completed sessions
- Payments
- Remaining sessions
- Renewal timing
- Quick mobile updates

## How to use it

### Step 1: Add your clients

Open the `Clients` tab and enter each client's name, contact details, goal, and next check-in date.

### Step 2: Add session packages

Open the `Packages` tab and enter the package each client purchased.

Important columns:

- `Sessions Purchased`
- `Price`
- `Amount Paid`
- `Renewal Threshold`

### Step 3: Log completed sessions

Open the `Sessions` tab and add each session.

Set `Completed` to `Yes` only when the session is completed.

### Step 4: Log payments

Open the `Payments` tab and enter each payment received.

### Step 5: Check the Dashboard

Use the `Dashboard` tab to see:

- Active clients
- Total revenue
- Unpaid balance
- Clients who need renewal
- Remaining sessions by package

### Step 6: Use Mobile Input after sessions

The `Mobile Input` tab is a simple narrow table for quick updates from your phone.

Use it to capture:

- Session completed
- Payment received
- Next session date
- Notes

Then copy the information into the main tabs when you are back at your computer.

## Important note

This is a spreadsheet template. It does not send reminders or emails automatically.
