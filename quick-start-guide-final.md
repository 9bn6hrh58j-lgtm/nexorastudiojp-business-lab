# Personal Trainer Client, Session & Payment Tracker

## Quick Start Guide

This tracker helps personal trainers and fitness coaches manage clients, session packages, completed sessions, payments, remaining sessions, and renewal timing.

## Step 1: Add Your Clients

Open the `Clients` tab.

Enter each client's name, email, phone number, status, goal, start date, and next check-in date.

## Step 2: Add Session Packages

Open the `Packages` tab.

Enter the package each client purchased.

Important columns:

- Sessions Purchased
- Price
- Amount Paid
- Payment Status
- Renewal Threshold

## Step 3: Log Completed Sessions

Open the `Sessions` tab.

Add each session after it is completed.

Set `Completed` to `Yes` only when the session is finished.

## Step 4: Record Payments

Open the `Payments` tab.

Enter the payment date, amount, payment method, and notes.

## Step 5: Check The Dashboard

Open the `Dashboard` tab to review:

- Active clients
- Total package revenue
- Total payments received
- Unpaid balance
- Completed sessions
- Packages needing renewal

## Step 6: Use Mobile Input

Use the `Mobile Input` tab for quick phone-friendly updates after a session.

You can record:

- Session completed
- Payment received
- Next session date
- Notes

## Step 7: Review Renewals

Open the `Renewal Checklist` tab.

Use it to check which clients are running low on sessions and should receive a renewal offer.

## Important Note

This is a spreadsheet template. It does not send automatic emails, text messages, or reminders.
