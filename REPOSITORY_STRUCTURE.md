## Why Files Are Excluded

| File or folder | Reason |
|---|---|
| `personal-trainer-session-tracker.xlsx` | Finished paid product |
| `quick-start-guide.pdf` | Paid product support file |
| `images/` | Etsy listing images |
| `image-sources/` | Product image source files |
| `etsy-*.md` | Etsy sales copy and launch assets |
| `READY_TO_LAUNCH.md` | Private launch operations |
| `WAITING_PERIOD_TASKS.md` | Payoneer and launch waiting-period notes |
| `POST_LAUNCH_CHECKLIST.md` | Private sales operations |
| `sales-experiment-plan.md` | Private sales strategy |
| `HIGH_TICKET_MARKET_RESEARCH.md` | Private business research |
| `BOARD_REPORT.md` | Private management report |

## Suggested Public Repository Purpose

Public description:

```text
A sample CSV-to-spreadsheet workflow for creating a lightweight client/session/payment tracker.
```

Do not describe it as an Etsy product or include live sales strategy in the public repository.
